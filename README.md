# Nganyalicious
A platform embedded on WhatsApp and/or available via SMS/USSD that makes the Kenyan public transport system more accessible, opening up more of the country to the average mwananchi.

####################################################################################
First, get publicly available data on licensed PSV Operators and their routes. 
Sources include: 
1. Official Sources (Most Accurate)
   • NTSA Website & Public Notices
   • Kenya Gazette
   • NTSA TIMS (Transport Integrated Management System)
   • NTSA Mobile Application
2. Digital and Social Media Updates
   • Tuko.co.ke
   • Maudhui House/Sacco Review
   • NTSA Twitter/X Handle

####################################################################################
Next, map out some Sacco routes and their stops, potentially using Google Maps API. 

####################################################################################
Develop an algorithm to help a user find the easiest/most convenient way from their point of origin to their desired destinations, making the appropriate connections between PSV Operators.
